## 主动技能

### 苍蓝火环

激活环绕周身的苍蓝火环：

* 持续消耗魔力。
* 对周围敌人造成伤害并施加**剧烈燃烧**效果。
* 火环激活时，**灵魂收割效果极大强化**（击杀额外回血+回魔）。

### 狐火吐息

消耗大量魔力，向前方喷吐出毁灭性的**灵魂狐火**：

* 造成瞬间伤害。
* 长时间点燃敌人。
* **灵魂火**：无法被水扑灭。

### 治疗术

物品栏主动技能，消耗魔力治疗自身或友方。

---

## 被动天赋

### 灵魂收割

作为灵体生物，可从消逝的生命中汲取力量：

* 击杀目标恢复大量魔力。
* 苍蓝火环激活时，击杀额外恢复生命值与能量。

### 魔力汲取

* 直接从生物体内抽取生命精华，获得能量与饱食度。
* 缓慢吸收诅咒之月残留在大气中的能量。

### 灵体化身

* 不再是单纯血肉之躯，免疫多种负面状态。
* 但**有益的药水效果也会消失**。
* 能量充足时，溢出的魔力会构成一层**魔力护盾**，减免少量伤害。

### 继承能力

* 透视生物。
* 在甜浆果丛中穿梭自如。

## 弱点与限制

| 限制 | 说明 |
| --- | --- |
| 魔力依赖加剧 | 魔力过低时不仅虚弱，还会**增加受到的伤害** |
| 魔力过载风险 | 存储已近极限，过度吸收魔力可能像气球一样**爆炸** |
| 脆弱体质 | 物理生命依然偏低，但比普通使魔稍能承受伤害 |
| 护甲限制 | 无法装备腿甲与鞋 |
| 水域行动 | 在水中行动迟缓 |
| 饱腹效率 | 从食物中获取的饱腹感更少 |
