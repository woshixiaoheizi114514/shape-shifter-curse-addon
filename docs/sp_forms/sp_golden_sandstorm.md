# 金沙岚

**Golden Sandstorm SP**

> "当诅咒与金色沙暴完全融合，你便不再仅仅是沙漠中的胡狼……
> 你是荒漠的回响，侵蚀之砂的化身。"
>
> 专属饰品/工具：蚀沙棱晶、枯沙指环

你的身躯被金色沙尘与凋零气息缠绕，毛发间流动着像日暮沙丘般的暗金光泽。你踏过的地方仿佛残留着细碎的魂砂，双眼中映着一场永不止息的沙暴。

## 主动技能

### 凋零金沙（蓄力 AOE）

蓄力 **1 秒** 后释放 **15 格范围** 的金沙风暴：

* 使非白名单生物陷入**沙盲 3 秒**。
* 为目标叠加 **1 层侵蚀烙印**。
* 蓄力期间**移动速度降低 50%**。
* 受到伤害会**打断蓄力**，并进入 **7 秒** 短冷却。
* 成功释放后进入 **26 秒** 冷却。

### 引爆标记

引爆所有**非绿色状态**的侵蚀烙印目标：

* 造成目标**当前生命值 20%** 的伤害（上限 **20 点**）。
* 回复自身 **20% 已损生命**。
* **3 层目标会先触发被动爆发再被引爆**（双重收益）。

## 被动天赋

### 侵蚀烙印（核心被动）

近战攻击会为目标叠加侵蚀烙印：

* **最多 3 层**，持续 **10 秒**。
* 同一目标默认每秒最多叠加 1 层。
* **3 层后再次攻击**触发**被动爆发**：
  * 造成目标当前生命值 **20%** 的伤害（上限 20）。
  * 回复自身 **10% 最大生命值**。
* 触发后目标进入**绿色冷却状态**，短时间内无法继续叠层。

### 烙印感知（视觉辅助）

被侵蚀烙印标记的生物会根据层数显示不同颜色的发光轮廓：

| 颜色 | 含义 |
| --- | --- |
| 黄色 | 第 1 层 |
| 橙色 | 第 2 层 |
| 红色 | 第 3 层（满层，下次攻击触发爆发） |
| 绿色 | 叠层冷却中 |

### 反噬冲击

受到攻击时自动释放反噬：

* 对周围 **4 格内**非白名单生物造成击退。
* 施加**凋零 I** 效果 5 秒。

### 凋零适应

免疫 **凋零、失明、沙盲** 效果。

### 继承自三进化阿努比斯之狼

灵魂疾行 III、亡灵不死、凋零传播攻击与吸血等大部分亡灵本能。

## 生命恢复机制

金沙岚**无法依靠饱食度自然回血**，且生命恢复药水无效。生命只能通过以下方式获得：

| 来源 | 回复量 |
| --- | --- |
| 你施加的凋零每次对敌造成伤害 | 半颗心 |
| 亲手击杀敌人 | 2 颗心 |
| 冥狼击杀敌人 | 1.5 颗心 |
| 被动调息（未交战） | 每 10 秒半颗心 |
| 被动调息（交战中） | 每 6 秒半颗心 |

| 限制 | 说明 |

## 战斗定位

**烙印爆发型近战**

核心循环：

1. **凋零金沙**：先用蓄力 AOE 给一片敌人挂上第 1 层烙印 + 沙盲。
2. **近战叠层**：手动近战推到 3 层，触发**被动爆发**（20% 真伤 + 10% 回血）。
3. **引爆标记**：群体引爆所有 1/2 层烙印目标（20% 真伤 + 20% 已损回血）。
4. **反噬冲击**：受击时自动反推 + 凋零，保命自卫。

适合在中等密度的中血量敌群中进行**滚雪球式收割**。
