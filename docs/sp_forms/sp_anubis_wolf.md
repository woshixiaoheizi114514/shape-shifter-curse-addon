## 主动技能

### 死亡领域（核心控场）

蓄力 **2 秒** 后：

* 将周围 **16 格半径** 的圆柱范围内所有方块**暂时变为灵魂沙**。
* 领域内敌对生物受到 **缓慢 I**、**凋零 I**，并被削减 **15% 生命值上限**。
* 领域持续 **15 秒**。

> 受白名单保护的玩家/宠物方块不会被替换。

### 冥狼裁庭（召唤）

嚎叫 **1.5 秒** 后召唤冥狼仆从：

* 冥狼攻击时附加凋零效果。
* 攻击已凋零的目标时**治愈施法者**。
* **死亡领域激活时**：召唤数量翻倍，且冥狼获得增强属性。

---

## 被动天赋

### 灵魂审视

你的双眼能洞察死亡的气息：

* **16 格内**生命值**低于 20%** 的生物会在你的视野中**发光**并散发灵魂粒子。
* 无处遁形。

### 死亡宣告

对生命值**低于 30%** 的目标造成的伤害提高 **20%**。垂死的猎物无法逃脱审判。

### 凋零追猎

当 **24 格内**存在被凋零影响的生物时：

* 在**非灵魂沙区域**获得 **20% 移动速度**加成。
* 凋零的气息驱使裁决者加速追击。

### 亡灵中立

* 作为亡灵之力的化身，**亡灵生物不再主动攻击你**，除非你先发起攻击。
* 停止攻击 **30 秒**后中立关系恢复。

### 灵魂沙回复

站在**灵魂沙或灵魂土**上时，冥界的能量为你缓慢恢复生命值。

### 继承自三进化阿努比斯之狼

* 灵魂疾行 III
* 亡灵不死
* 凋零传播攻击与吸血
* 瞬移等其它三进化亡灵本能

## 弱点与限制

| 限制 | 说明 |
| --- | --- |
| 受击凋零 | 受到攻击时激发的亡灵力量会让你陷入凋零状态 |
| 无自然再生 | 生命恢复药水对你无效，从饱和度获得的生命恢复极为缓慢 |
| 食物腐烂 | 击杀生物掉落的肉类会腐烂为腐肉，收获的作物有概率化为沙尘 |
| 素食无效 | 素食无法为你提供任何饱和度 |
| 对亡灵削弱 | 你对亡灵生物的直接伤害有所下降 |
| 护甲限制 | 普通腿甲与鞋不再适合你们四足身体结构 |

## 战斗定位

**控场召唤师 / 区域改造者**

死亡领域是核心技能：把任何战场临时转化为灵魂沙地，对所有敌人挂凋零 + 减速 + 削血。配合冥狼裁庭，能在 15 秒窗口内打出**双倍数量 + 增强属性的冥狼海**进行碾压。

适合长时间消耗战、Boss 战、PVP 团战。
