# 契灵 —— 禁忌兵器

**Familiar Fox · Mancianima / Spirit Realm Lord（灵界之主）**

> "契灵不是神明的赐福，而是从神明尸骸中撕下的答案……
> 女巫以进化石焊接灵魂，将使魔锻成劫掠阵营最沉默、最昂贵的禁忌兵器。"

它仍保留使魔的轮廓，但毛色、瞳光与灵压都像被灵界结晶重新校准过。它的动作冷静、精密，像一件经历过无数次稳定化仪式后终于醒来的活体兵器。

## 主动技能

### 契灵烙印（主技能）

持续瞄准生物即可进行兵器校准：

1. **第一段（黄色烙印）**：消耗 **15 法力**，对 32 格内瞄准的目标打下黄色烙印；首段冷却 5 秒。
2. **第二段（红色烙印）**：升级为红色烙印，施加 **恐惧 + 凋零音效**，1 秒一次推进间隔。
3. **结算（2 秒引导真伤）**：在烙印窗口内对目标完成 2 秒引导，造成 **20% 目标剩余生命的真实伤害**，单次上限 **20 点**；命中后 **+15 秒** 追击系统冷却。
4. **空触发**：准星无目标时退化为 **5 法力** 的空触发，避免误按浪费完整冷却。

被烙印的目标只有契灵自己客户端可见**发光描边**（黄 / 橙 / 红，对应当前阶段），不会暴露给队友或其他形态。

### 魂跃（次要技能）

短距离传送：消耗 **15 法力**，CD **5 秒**。可在客户端配置中选择以下两种模式：

* **直线传送（Raycast）**：朝准星方向直线传送，最远 **8 格**，遇到方块停下。
* **平台锁定（Platform）**：按住技能键时显示落点的紫色粒子预览（仅自己可见），松开后传送；如果落点无可站立平台，会拒绝传送且**不消耗 CD/法力**。

> 命中契灵烙印锁定的目标后击杀，会重置传送冷却并补满抗伤加成。

### 敲钟袭击 / 村民榨取

在村庄内敲响钟会触发契灵袭击：

* 附近铁傀儡会**转而追杀你**。
* 出现独立的剩余村民 boss 条；清除范围内所有村民后获得奖励。
* 冷却为 **1 MC 天**。

不适合长期居住在村庄附近的玩家。

## 被动天赋

### 契灵法力系统

独立于其它形态的**专属法力条**：

* 独立的法力上限与回复曲线。
* 受击或主动技施放后 **5 秒回复暂停**。
* HUD 上方显示抗性条：黄/红标层级 + 引导进度同步显示。

### 兵器化身体调律

* **徒手攻击距离扩展**：契灵的爪牙判定距离比普通形态更远。
* **武器攻击速度 / 距离适配**：携带武器时同步放宽攻击节奏与判定范围。
* **负面效果免疫**：作为高度稳定化的灵体，免疫多种负面状态。
* **自定义无敌帧 / 受击 i-frames**：被击中后短暂获得保护帧，避免被多段连击秒杀。

### 村民掠夺

* **无法**与村民或流浪商人交易。
* 击杀村民 / 流浪商人时，按其交易内容掠夺物品，并有 **30%** 几率额外掉落 1-3 个绿宝石。
* 附近村民与流浪商人本能逃离你，并被自动施加减速。
* 远程投射物追踪 mixin：契灵的远程攻击会更精准地追踪逃跑目标。

### 魔力汲取

* 击杀目标时从生命消逝中回收魔力。
* 缓慢吸收诅咒之月残留在大气中的能量。

### 继承自使魔的能力

* 透视生物（标记态目标自动发光，仅自己可见）。
* 在甜浆果丛中穿梭自如。
* 轻便的体型与敏捷的身段。

## 弱点与限制

| 限制 | 说明 |
| --- | --- |
| 魔力依赖 | 所有主动技都依赖魔力驱动；魔力不足时无法传送，烙印推进也会变慢 |
| 高压灵能 | 进化石带来的魔力并非自然流动，必须通过杀戮与汲取维持战斗循环 |
| 阵营烙印 | 被刻入劫掠阵营兵器本能，无法与村民/流浪商人交易 |
| 敲钟代价 | 敲钟会拉仇附近所有铁傀儡；冷却长达 1 MC 天 |
| 护甲限制 | 仍以使魔为基底，无法装备腿甲与鞋 |
| 水域行动 | 在水中行动迟缓 |
| 饱腹效率 | 从食物中获得的饱腹感更少，单靠普通进食难以支撑改造过的灵魂躯壳 |

## 战斗定位

**单体猎杀型兵器**

契灵的核心循环是：

1. 用 **黄→红烙印** 锁定目标。
2. 启动 **2 秒引导真伤** 完成清算。
3. 用 **魂跃** 切目标或追逃。

与团战为主的灵界之主（SP 使魔）不同，契灵更适合**专注一个高血量目标**：真伤上限 20 点对中高血量 boss / 玩家 / 铁傀儡都有可观收益。
