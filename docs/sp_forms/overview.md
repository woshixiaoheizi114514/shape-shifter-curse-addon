# SP 形态概述

**SP 形态（Special Form）** 是 SSCA 的核心进阶形态体系。它们不是简单数值增强，而是为原形态重写战斗循环：有的围绕资源条，有的依赖召唤物，有的专注控场、隐身、治疗或百分比爆发。

***

## 进化方式

### 月髓十字环

月髓十字环路线需要处于**被诅咒之月的夜晚**，并且当前形态满足对应基础形态要求。

| 基础形态 | 目标形态 |
| ---- | ---- |
| 使魔红狐三阶段 | [SP使魔（灵界之主）](sp_familiar_fox.md) |
| 美西螈三阶段 | [SP美西螈（深渊霸主）](sp_axolotl.md) |
| 原版悦灵形态 | [SP堕落悦灵（堕落之灵）](sp_fallen_allay.md) |
| 阿努比斯之狼三阶段 | [SP阿努比斯（冥裁者）](sp_anubis_wolf.md) |

红不是稳定的月髓十字环目标形态，而是 SP 使魔在被诅咒之月中以 **5%**几率限时转化，持续 **12000 tick（10 分钟）**后回到 SP 使魔。

## 玩法分组

### 资源循环型

SP使魔、红、SP美西螈、SP雪狐、SP悦灵都强依赖资源条。魔力、湿润值、霜寒值和水晶能量决定它们能否连续作战。

### 控场与召唤型

SP雪狐、SP堕落悦灵、SP阿努比斯偏向改变战场：冰风暴吸附、恼鬼追击、死亡领域改造地形，都会迫使敌人改变站位。

### 刺杀与爆发型

SP野猫、契灵、金沙岚更偏向选定目标后完成爆发。它们的强度来自隐身先手、烙印推进或百分比结算，而不是无脑站桩。吸血蝙蝠则是典型的夜战近战猎手：靠血渴值越打越强，但白天与面对铁器时格外脆弱。

### 团队辅助型

SP悦灵的定位最接近纯辅助：净化、治疗、图腾守护与远程增伤让它在多人环境中价值很高，但自身生命很低，需要队友保护输出窗口。寄生果蝠则以“种子」运营为核心：给友方挂增益果、给敌方挂削弱果，并用感染孢子弹提供持续的群体压制与团队治疗。

## 白名单说明

多数范围伤害、控制、治疗与召唤物索敌都接入了 SSCA 白名单逻辑：受保护目标不会被误伤，增益技能也会按增益白名单判断可治疗对象。不同技能的边界略有差异，具体以各形态页面的技能说明为准。
