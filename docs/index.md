## 关于本模组

**幻形者诅咒扩展包** 是 [幻形者诅咒 (Shape Shifter Curse)](https://www.mcmod.cn/class/20760.html) 的附属模组，基于 Fabric 开发，为原版形态添加了全新的 **SP（特殊）形态** 变体，每种变体都拥有重构的主动技能、被动天赋和独特的战斗风格。

## 核心玩法

> "诅咒的本质并非毁灭，而是进化……当月光被更深邃的暗影吞没，唯有敢于拥抱极端力量的幻形者，才能触及那禁忌的形态。"

不再满足于普通的野兽本能？是时候寻求突破了！

**进化之钥**：使用 [月髓十字环](items/evolution_items.md#月髓十字环) 或 [进化石](items/evolution_items.md#进化石)，在对应条件下让你的形态进化为更高级的 SP 形态，感受魔力的重塑与融合。

## 模组特色

- 各个 SP 形态拥有完全不同的主动技能与被动天赋
- 专为 **PVP** 进行的平衡设计，每种形态都有独特的战斗风格
- 有依靠小体型在人群中消耗的，有靠隐身接近刺杀的……
- 各形态都有属于自己的 **专属饰品**
- 丰富的背景故事与世界观设定
